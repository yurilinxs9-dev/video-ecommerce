# Story 2.1 — VideoCarousel: Estrutura Base com Swiper

**Epic:** 2 — Carrossel de Vídeos
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 2-3h

---

## User Story

Como usuário da loja,
Quero ver um carrossel de vídeos verticais centralizado e interativo,
Para descobrir produtos de forma visual e engajante.

---

## Contexto

Componente central do widget. Usa Swiper.js 11+ com React wrapper. O slide ativo fica centralizado, levemente maior, com slides laterais parcialmente visíveis.

**Depende de:** Story 1.1 (setup), 1.2 (tipos), 1.3 (utils)

## Notas Técnicas

### VideoCarousel.tsx — Config Swiper Completa
```tsx
import { Swiper, SwiperSlide } from 'swiper/react';
import { Navigation, Pagination, Autoplay } from 'swiper/modules';
import 'swiper/css';

<Swiper
  modules={[Navigation, Pagination, Autoplay]}
  slidesPerView="auto"
  centeredSlides={true}
  loop={true}
  loopAdditionalSlides={videos.length}
  initialSlide={Math.floor(videos.length / 2)}
  speed={550}
  spaceBetween={10}
  grabCursor={true}
  watchSlidesProgress={true}
  navigation={{ prevEl: prevRef.current, nextEl: nextRef.current }}
  pagination={{ el: paginationRef.current, clickable: true }}
  onSlideChange={(swiper) => onSlideChange(swiper.realIndex)}
>
  {videos.map((video, index) => (
    <SwiperSlide key={video.id} style={{ width: 'var(--slide-width)' }}>
      <VideoSlide video={video} isActive={index === activeIndex} />
    </SwiperSlide>
  ))}
</Swiper>
```

### CSS para scale/opacity (via CSS custom ou inline style)
```css
:root { --slide-width: 170px; }
@media (min-width: 769px) { :root { --slide-width: 205px; } }
@media (min-width: 1025px) { :root { --slide-width: 240px; } }

.swiper-slide {
  transition: transform 500ms cubic-bezier(.22,.68,0,1), opacity 500ms;
  transform: scale(0.92);
  opacity: 0.72;
  border-radius: 14px;
  overflow: hidden;
}
.swiper-slide-active {
  transform: scale(1);
  opacity: 1;
  box-shadow: 0 6px 24px rgba(0,0,0,.13);
}
.swiper-slide-prev, .swiper-slide-next { opacity: 0.82; }
```
