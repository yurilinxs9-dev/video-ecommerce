# Story 3.5 — FullscreenProduct: Card e Botão

**Epic:** 3 — Player Fullscreen
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1h

---

## User Story

Como usuário,
Quero ver o produto associado ao vídeo e poder acessá-lo diretamente,
Para comprar com um clique após assistir ao vídeo.

---

## Acceptance Criteria

- [ ] **AC-1:** Posição: `absolute bottom: 0`, full width
- [ ] **AC-2:** Gradient overlay: `linear-gradient(transparent, rgba(0,0,0,.78))`
- [ ] **AC-3:** Container produto: flex, gap 10px, padding 10px, `background: rgba(255,255,255,.1)`, border-radius 9px, `backdrop-filter: blur(4px)`
- [ ] **AC-4:** Thumbnail: 44x44px, border-radius 6px
- [ ] **AC-5:** Nome: 12px semibold, branco, ellipsis
- [ ] **AC-6:** Preço: 13px bold, branco
- [ ] **AC-7:** Botão "Adicionar produto": full width, background branco, font bold 13px, border-radius 8px
- [ ] **AC-8:** Clique no botão → `window.location.href = product.url`
- [ ] **AC-9:** Margem bottom: 80px (espaço para gestos de navegação)
