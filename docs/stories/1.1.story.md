# Story 1.1 — Setup Inicial: Next.js + Tailwind + Swiper

**Epic:** 1 — Fundação do Projeto
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1-2h

---

## User Story

Como desenvolvedor,
Quero um projeto Next.js 14 configurado com Tailwind CSS e Swiper.js,
Para ter a base técnica pronta para desenvolver o widget.

---

## Contexto

Este é o primeiro passo. O repositório `video-commerce-widget` já existe com AIOX framework. Precisamos inicializar o projeto Next.js com a stack correta dentro desse repositório (ou em subpasta dedicada se necessário).

**Stack:**
- Next.js 14+ (App Router)
- React 18+
- Tailwind CSS 3+
- Swiper.js 11+ (`swiper/react`)
- TypeScript

## Notas Técnicas

### next.config.js (Swiper ESM fix)
```javascript
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['swiper'],
}
module.exports = nextConfig
```

### tailwind.config.js
```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  content: ['./src/**/*.{js,ts,jsx,tsx,mdx}'],
  theme: { extend: {} },
  plugins: [],
}
```

### tsconfig.json paths
```json
{
  "compilerOptions": {
    "paths": { "@/*": ["./src/*"] }
  }
}
```
