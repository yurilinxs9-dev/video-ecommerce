# Story 6.8 — Admin: Configurações Gerais

**Epic:** 6 — Painel de Administração
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1.5h

---

## User Story

Como lojista,
Quero configurar as opções gerais do widget sem editar código,
Para personalizar cores, WhatsApp padrão e quais funcionalidades exibir.

---

## Acceptance Criteria

- [ ] **AC-1:** Rota `/admin/settings` com formulário de configurações
- [ ] **AC-2:** Link "Configurações" no header do admin
- [ ] **AC-3:** Campo: Cor de destaque (accent) — input `type="color"` + preview hex
- [ ] **AC-4:** Campo: WhatsApp padrão — input texto com placeholder e hint
- [ ] **AC-5:** Campo: URL base da loja — input URL
- [ ] **AC-6:** Toggles (checkboxes ou switches): Mostrar Setas, Mostrar Dots, Mostrar Curtir, Mostrar WhatsApp, Mostrar Compartilhar
- [ ] **AC-7:** Campo: Delay do autoplay (ms) — input number, min 3000, max 30000
- [ ] **AC-8:** Toggle: Autoplay ativado
- [ ] **AC-9:** Botão "Salvar Configurações" → `PUT /api/settings` → toast de sucesso
- [ ] **AC-10:** Formulário pré-preenchido com valores atuais via `GET /api/settings`
