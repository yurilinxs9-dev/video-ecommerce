# Story 6.5 — Admin Dashboard: Lista de Vídeos

**Epic:** 6 — Painel de Administração
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 2h

---

## User Story

Como lojista,
Quero ver todos os meus vídeos cadastrados em um painel organizado,
Para ter visibilidade do que está publicado e gerenciar facilmente.

---

## Acceptance Criteria

- [ ] **AC-1:** Rota `/admin` (index) exibe dashboard principal
- [ ] **AC-2:** Header com logo/título "Video Commerce Admin" + botão logout
- [ ] **AC-3:** Botão "Adicionar Vídeo" → navega para formulário
- [ ] **AC-4:** Lista de vídeos em cards ou tabela com: thumbnail (60x80px), nome do produto, preço, URL do vídeo (truncada)
- [ ] **AC-5:** Cada item tem botões: "Editar" e "Remover" (com confirmação)
- [ ] **AC-6:** Badge de posição/ordem do vídeo (número 1, 2, 3...)
- [ ] **AC-7:** Estado vazio: "Nenhum vídeo cadastrado. Adicione o primeiro!" com CTA
- [ ] **AC-8:** Feedback de ações: toast/alert de sucesso/erro ao remover
- [ ] **AC-9:** Design clean, responsivo, sem dependências CSS externas (Tailwind)

---

## Tasks

- [ ] Criar `src/app/admin/page.tsx` (dashboard)
- [ ] Criar `src/app/admin/layout.tsx` (layout compartilhado do admin com header)
