# Story 6.9 — Admin: Preview ao Vivo do Carrossel

**Epic:** 6 — Painel de Administração
**Status:** Draft
**Prioridade:** Should
**Estimativa:** 1.5h

---

## User Story

Como lojista,
Quero ver uma prévia do carrossel com os dados atuais diretamente no admin,
Para validar o visual antes de publicar, sem precisar ir até a loja.

---

## Acceptance Criteria

- [ ] **AC-1:** Seção "Preview" na página `/admin` abaixo da lista de vídeos
- [ ] **AC-2:** Preview renderiza o `VideoCarousel` real com os dados atuais do KV
- [ ] **AC-3:** Preview é somente visualização (não abre fullscreen, cliques desabilitados)
- [ ] **AC-4:** Escala reduzida: max-width 600px, centrado
- [ ] **AC-5:** Label "Preview do Widget" com badge "Ao vivo"
- [ ] **AC-6:** Botão "Atualizar Preview" para refetch dos dados
- [ ] **AC-7:** Quando lista de vídeos é reordenada → preview atualiza automaticamente
- [ ] **AC-8:** Container com `pointer-events: none` para desabilitar interações
