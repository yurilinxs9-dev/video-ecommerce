# Story 3.2 — ProgressBars Animadas

**Epic:** 3 — Player Fullscreen
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1.5h

---

## User Story

Como usuário,
Quero ver barras de progresso no topo do fullscreen como no Instagram Stories,
Para saber em qual vídeo estou e quanto falta.

---

## Acceptance Criteria

- [ ] **AC-1:** Uma barra por vídeo, layout flex row com gap de 3px
- [ ] **AC-2:** Barras anteriores: preenchidas (branco 100%)
- [ ] **AC-3:** Barra atual: animando de 0% a 100% pela duração real do vídeo
- [ ] **AC-4:** Barras futuras: fundo semi-transparente (branco 30%)
- [ ] **AC-5:** Altura: 2.5px, padding: 8px 10px
- [ ] **AC-6:** Animação CSS via `animation` / `keyframes` ou `transition` + `width`
- [ ] **AC-7:** Ao trocar de vídeo → barra anterior fica 100%, nova começa do zero
- [ ] **AC-8:** Props: `total`, `current`, `videoDuration`, `isPlaying`
