## Notas Técnicas

```json
// vercel.json
{
  "headers": [
    {
      "source": "/widget.js",
      "headers": [
        { "key": "Access-Control-Allow-Origin", "value": "*" },
        { "key": "Access-Control-Allow-Methods", "value": "GET" },
        { "key": "Cache-Control", "value": "public, max-age=86400" }
      ]
    }
  ]
}
```

### Cloudinary — Upload dos Vídeos
1. Criar conta em cloudinary.com (free: 25GB)
2. Upload dos vídeos MP4 verticais (1080x1920 ideal)
3. Upload dos posters (screenshot dos vídeos)
4. Copiar URLs diretas
5. Atualizar `videos.json` com as URLs

### Código de Embed para Bagy
```html

<script src="https://SEU-PROJETO.vercel.app/widget.js"></script>
```
