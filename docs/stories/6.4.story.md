# Story 6.4 — Widget Lê Dados da API em vez de JSON Estático

**Epic:** 6 — Painel de Administração
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1h

---

## User Story

Como lojista,
Quero que o widget na loja sempre exiba os vídeos mais atuais do painel admin,
Para que mudanças feitas no admin reflitam automaticamente na loja.

---

## Acceptance Criteria

- [ ] **AC-1:** `VideoShowcase.tsx` faz fetch de `/api/videos` e `/api/settings` em vez de importar JSON
- [ ] **AC-2:** Estado de loading: exibe skeleton ou spinner enquanto carrega
- [ ] **AC-3:** Estado de erro: falha silenciosa (não quebra a página) com fallback para JSON local
- [ ] **AC-4:** `mount.ts` (widget embed) também faz fetch da API (URL configurável)
- [ ] **AC-5:** URL da API configurável: `window.__VIDEO_WIDGET_API_URL__` ou default para mesmo domínio
- [ ] **AC-6:** Cache: `cache: 'no-store'` na produção ou `revalidate: 60` para ISR (configurável)
- [ ] **AC-7:** Dados retornados pela API têm o mesmo shape do `WidgetConfig`

## Notas Técnicas

```tsx
// VideoShowcase.tsx — fetch ao invés de import
export function VideoShowcase() {
  const [config, setConfig] = useState<WidgetConfig | null>(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const apiUrl = (window as any).__VIDEO_WIDGET_API_URL__ || '';

    Promise.all([
      fetch(`${apiUrl}/api/videos`).then(r => r.json()),
      fetch(`${apiUrl}/api/settings`).then(r => r.json()),
    ])
      .then(([videos, settings]) => {
        setConfig({ videos, settings });
      })
      .catch(() => {
        // Fallback para dados estáticos embutidos no bundle
        import('@/data/videos.json').then(data => setConfig(data as WidgetConfig));
      })
      .finally(() => setLoading(false));
  }, []);

  if (loading) return <CarouselSkeleton />;
  if (!config) return null;

  return <VideoCarousel config={config} />;
}
```

```tsx
// CarouselSkeleton — placeholder durante loading
function CarouselSkeleton() {
  return (
    <div style={{ display: 'flex', gap: 10, padding: '20px 0', overflow: 'hidden' }}>
      {[...Array(5)].map((_, i) => (
        <div key={i} style={{
          width: 170, height: 260, borderRadius: 14,
          background: 'linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%)',
          backgroundSize: '200% 100%',
          animation: 'shimmer 1.5s infinite',
          flexShrink: 0,
        }} />
      ))}
    </div>
  );
}
```
