# Story 6.6 — Admin: Formulário Adicionar/Editar Vídeo

**Epic:** 6 — Painel de Administração
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 2.5h

---

## User Story

Como lojista,
Quero um formulário para adicionar e editar vídeos com todos os campos necessários,
Para gerenciar meu conteúdo sem precisar editar arquivos JSON manualmente.

---

## Acceptance Criteria

- [ ] **AC-1:** Rota `/admin/videos/new` — formulário de criação
- [ ] **AC-2:** Rota `/admin/videos/[id]` — formulário de edição (pré-preenchido)
- [ ] **AC-3:** Campos obrigatórios: URL do Vídeo MP4, URL do Poster, Nome do Produto, Preço, URL da Imagem, URL da Página do Produto
- [ ] **AC-4:** Campo opcional: Número WhatsApp (com hint: "Deixe vazio para usar o padrão")
- [ ] **AC-5:** Validação client-side: campos obrigatórios, formato de URL
- [ ] **AC-6:** Preview inline: ao digitar URL do poster, exibe thumbnail preview
- [ ] **AC-7:** Botão "Salvar" → `POST /api/videos` (novo) ou `PUT /api/videos/[id]` (edição)
- [ ] **AC-8:** Após salvar com sucesso → redireciona para dashboard com toast "Vídeo salvo!"
- [ ] **AC-9:** Botão "Cancelar" → volta ao dashboard
- [ ] **AC-10:** ID gerado automaticamente: `video-{timestamp}` para novos vídeos
- [ ] **AC-11:** Campos com label, placeholder e hint text descritivo

// src/components/admin/VideoForm.tsx
interface VideoFormProps {
  initialData?: VideoItem;
  onSubmit: (data: VideoItem) => Promise<void>;
  isLoading?: boolean;
}

// Campos do formulário:
const fields = [
  { name: 'videoUrl', label: 'URL do Vídeo MP4', required: true,
    placeholder: 'https://res.cloudinary.com/seu-cloud/video/upload/v1/video.mp4',
    hint: 'URL direta do arquivo .mp4 no Cloudinary' },
  { name: 'posterUrl', label: 'URL do Poster (capa)', required: true,
    placeholder: 'https://res.cloudinary.com/seu-cloud/image/upload/v1/poster.jpg',
    hint: 'Imagem de capa exibida antes do vídeo carregar' },
  { name: 'product.name', label: 'Nome do Produto', required: true,
    placeholder: 'Camisa Transpasse Cintos' },
  { name: 'product.price', label: 'Preço', required: true,
    placeholder: 'R$ 299,99' },
  { name: 'product.image', label: 'URL da Imagem do Produto', required: true,
    placeholder: 'https://cdn.dooca.store/...' },
  { name: 'product.url', label: 'URL da Página do Produto', required: true,
    placeholder: 'https://sua-loja.com/produto' },
  { name: 'whatsapp', label: 'WhatsApp (opcional)', required: false,
    placeholder: '5537999999999',
    hint: 'Com DDI. Deixe vazio para usar o padrão das configurações' },
];

// Auto-gerar ID:
const generateId = () => `video-${Date.now()}`;
