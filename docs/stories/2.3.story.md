# Story 2.3 — ProductCard no Carrossel

**Epic:** 2 — Carrossel de Vídeos
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 1h

---

## User Story

Como usuário da loja,
Quero ver as informações do produto abaixo de cada vídeo,
Para identificar rapidamente nome, preço e poder acessar o produto.

---

## Acceptance Criteria

- [ ] **AC-1:** Layout: flex row, gap, padding `6px 8px`, background escuro semitransparente
- [ ] **AC-2:** Thumbnail produto: 40x40px, `border-radius: 6px`, `object-fit: cover`
- [ ] **AC-3:** Coluna com nome (truncado, `11.5px`, branco) e preço (bold, `12.5px`, branco)
- [ ] **AC-4:** Ícone de sacola: círculo com `accentColor`, 26x26px, SVG bag inside
- [ ] **AC-5:** Clique em qualquer parte do card → `window.open(product.url, '_blank')` ou `window.location.href`
- [ ] **AC-6:** Nome truncado com ellipsis quando muito longo
