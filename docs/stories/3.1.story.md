# Story 3.1 — FullscreenPlayer: Estrutura e Overlay

**Epic:** 3 — Player Fullscreen
**Status:** Draft
**Prioridade:** Must
**Estimativa:** 2-3h

---

## User Story

Como usuário da loja,
Quero assistir ao vídeo em formato fullscreen stories ao clicar no slide,
Para ter uma experiência imersiva de video commerce.

---

## Acceptance Criteria

- [ ] **AC-1:** Overlay fullscreen com `position: fixed`, `inset: 0`, `z-index: 9999`
- [ ] **AC-2:** Fundo: `rgba(0,0,0,0.9)` com `backdrop-filter: blur(4px)`
- [ ] **AC-3:** Container centralizado: max-width 400px desktop, 100% mobile
- [ ] **AC-4:** `border-radius: 14px` desktop, 0 mobile
- [ ] **AC-5:** Vídeo com som ativado (não mutado), autoplay ao abrir
- [ ] **AC-6:** `object-fit: cover`, `object-position: center top`
- [ ] **AC-7:** Botão X (top-right): 32x32px, `background: rgba(0,0,0,.3)`, fecha o fullscreen
- [ ] **AC-8:** Ao fechar: retoma autoplay do carrossel
- [ ] **AC-9:** `useFullscreen.ts` hook gerencia estado e callbacks
